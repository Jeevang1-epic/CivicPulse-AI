```txt
Continue CivicPulse AI.

Implement the dashboard and community intelligence layer.

Requirements:
1. Dashboard KPI cards:
   - total reports
   - open reports
   - urgent/critical reports
   - resolved reports
2. Priority queue sorted by severity desc and createdAt desc.
3. Filters:
   - status
   - category
   - severity
4. Status update controls:
   - open
   - in_review
   - assigned
   - resolved
5. Add AI/admin action plan drawer or panel.
6. Add neighborhood brief:
   - summarize top categories, urgent issues, and recommended focus
   - use Gemini if available
   - fallback deterministic summary if not
7. Improve landing page with strong pitch and architecture section.
8. Add mobile polish and loading skeletons.

Acceptance:
- Dashboard feels like a real operating system.
- Status update changes UI.
- Brief works with or without Gemini.
- Build passes.
```
